# sites-wanbo-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能包含笔记、工具页面、演示页面或其它类型的前端静态内容。仓库本身不针对任何特定域名或网站，仅作为个人或协作场景下的静态页面集合存储与分发使用。

## 目录说明

```
sites-wanbo-notes/
├── index.html          # 仓库页面索引（可选）
├── page-xxx/           # 每个页面独立目录
│   ├── index.html
│   └── assets/         # 页面相关资源（CSS, JS, 图片等）
├── ...
└── README.md           # 本文件
```

每个子目录对应一个独立的 HTML 页面项目，目录名通常反映页面用途或内容主题。资源文件建议存放在对应目录下的 `assets` 文件夹中，便于管理与引用。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面之间相互独立，不共享数据或状态。
- 归档内容可能不定期更新、增删或调整目录结构。
- 部分页面可能包含实验性功能或临时内容，不代表最终版本。
